# Z.E.U.S. — Zero-latency Executive Universal System

An Iron Man HUD style personal AI assistant powered by Claude API.

## Features
- Iron Man HUD UI (cyan/dark theme, hex grid, arc reactor)
- Voice input + voice output
- 3D spinning globe (Three.js)
- Adaptive personality engine
- Live weather + news
- Memory across sessions
- PWA for iPhone 15

## Tech Stack
- React
- Claude API (claude-sonnet-4-20250514)
- Three.js
- Web Speech API
- Vercel
